# 🎮 Number Guessing Game (C++)

This is a simple C++ console-based game where the player has to guess a randomly generated number.

## 🚀 Features
- Random number generation (1–100)
- Hints: Too High / Too Low
- Limited attempts (Game Over system)
- Displays number of attempts

## 🧠 Concepts Used
- Loops (while)
- Conditional statements (if-else)
- Random numbers (rand, srand)
- User input/output

## ▶️ How to Run
1. Compile the code using a C++ compiler
2. Run the program in terminal
3. Enter your guesses and try to win!
